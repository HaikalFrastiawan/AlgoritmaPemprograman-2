1. Pengenalan Nested IF (IF Bersarang)

Nested IF atau IF Bersarang adalah sebuah struktur logika di mana sebuah pernyataan IF berada di dalam pernyataan IF lainnya. Struktur ini memungkinkan kita untuk membangun alur pengecekan kondisi yang lebih kompleks dan bertingkat, terutama untuk mengevaluasi beberapa kondisi yang saling bergantung.

2. Logika AND

Logika AND hanya akan menghasilkan nilai True jika semua kondisi yang dievaluasi bernilai True. Jika ada satu saja kondisi yang False, maka hasil akhirnya adalah False.

Tabel Kebenaran Logika AND

TrueTrueTrue

FalseTrueFalse
FalseFalseFalse

3. Logika OR

Logika OR akan menghasilkan nilai True jika salah satu atau semua kondisi bernilai True. Hasilnya akan False hanya jika semua kondisi bernilai False.

Tabel Kebenaran Logika OR

Kondisi 1Kondisi 2Hasil
TrueTrueTrue

FalseTrueTrue
FalseFalseFalse

4. Konsep Pruning (Pemangkasan)

Pruning (atau short-circuit evaluation) adalah konsep di mana program berhenti mengevaluasi sisa kondisi jika hasil akhirnya sudah dapat ditentukan. Ini membuat eksekusi program lebih efisien.

Pruning untuk Logika AND

Pada logika AND, jika kondisi pertama sudah False, maka hasil akhir pasti False. Program tidak perlu memeriksa kondisi kedua.

5. Contoh Kasus NESTED IF: Logika AND

Studi Kasus: Seseorang berhak mendapatkan Surat Izin Mengemudi (SIM) jika usianya minimal 17 tahun DAN sudah lulus ujian teori.

Program Tanpa Pruning

Pendekatan ini menggunakan struktur Nested IF yang lebih kompleks untuk mencoba menampilkan semua kemungkinan hasil pengecekan. Perhatikan bagaimana alur logika menjadi kurang intuitif dibandingkan dengan pendekatan pruning.

# Program Tanpa prunning
usia = 17
lulus_ujian_teori = False

print("\n--- Pengecekan Kelayakan (Tanpa Pruning) ---")
print(f"Usia Anda: {usia} tahun")
print(f"Status Ujian Teori: {'Lulus' if lulus_ujian_teori else 'Tidak Lulus'}")
print("------------------------------------------")

# Pengecekan kondisi 1 (usia)
if usia >= 17:
  if lulus_ujian_teori == True:
    print("‚úÖ Pengecekan 1: Usia terpenuhi.")
    print("‚úÖ Pengecekan 2: Status lulus ujian terpenuhi.")
    print("\nüéâ Selamat! Anda berhak membuat SIM.")
  else:
    print("‚úÖ Pengecekan 1: Usia terpenuhi.")
    print("‚ùå Pengecekan 2: Status lulus ujian TIDAK terpenuhi.")
    print("\nMaaf, Anda tidak memenuhi semua syarat.")
else:
  if lulus_ujian_teori == True:
    print("‚ùå Pengecekan 1: Usia TIDAK terpenuhi.")
    print("‚úÖ Pengecekan 2: Status lulus ujian terpenuhi.")
    print("\nMaaf, Anda tidak memenuhi semua syarat.")
  else:
    print("‚ùå Pengecekan 1: Usia TIDAK terpenuhi.")
    print("‚ùå Pengecekan 2: Status lulus ujian TIDAK terpenuhi.")
    print("\nMaaf, Anda tidak memenuhi semua syarat.")

Program Menggunakan Pruning

Berikut adalah contoh struktur Nested IF yang menerapkan pruning. Jika `usia` kurang dari 17, blok `if` bagian dalam tidak akan pernah dieksekusi.

# Program Dengan prunning
usia = 16
lulus_ujian_teori = True

print("\n--- Pengecekan Kelayakan (Dengan Pruning) ---")
print(f"Usia Anda: {usia} tahun")
print(f"Status Ujian Teori: {'Lulus' if lulus_ujian_teori else 'Tidak Lulus'}")
print("------------------------------------------")

# Pengecekan kondisi 1 (usia)
if usia >= 17:
    print("‚úÖ Pengecekan 1: Usia terpenuhi.")
    # Jika usia terpenuhi, lanjut ke pengecekan 2
    if lulus_ujian_teori == True:
        print("‚úÖ Pengecekan 2: Status lulus ujian terpenuhi.")
        print("\nüéâ Selamat! Anda berhak membuat SIM.")
    else:
        print("‚ùå Pengecekan 2: Status lulus ujian TIDAK terpenuhi.")
        print("\nMaaf, Anda tidak memenuhi syarat karena belum lulus ujian.")
else:
    # Jika usia tidak terpenuhi, proses berhenti (pruning)
    print("‚ùå Pengecekan 1: Usia TIDAK terpenuhi.")
    print("\nMaaf, Anda tidak memenuhi syarat karena usia belum mencukupi.")

6. Contoh Kasus NESTED IF: Logika OR

Studi Kasus: Seseorang dianggap "layak menerima bantuan sosial" jika usianya minimal 60 tahun (dianggap lansia) ATAU memiliki surat rekomendasi khusus (misalnya dari dinas sosial karena kondisi tertentu).

Tabel Kebenaran Tanpa Pruning

Usia >= 60?Punya Rekomendasi?Hasil
TrueTrueLayak
TrueFalseLayak
FalseTrueLayak
FalseFalseTidak Layak

Program Tanpa Pruning

Pendekatan ini menggunakan struktur Nested IF yang lebih kompleks untuk menampilkan semua kemungkinan hasil pengecekan, sama seperti pada contoh logika AND.

# Inisialisasi variabel
usia = 55
punya_rekomendasi = True

print("\n--- Pengecekan Kelayakan (Tanpa Pruning) ---")
print(f"Usia Anda: {usia} tahun")
print(f"Punya Rekomendasi: {'Ya' if punya_rekomendasi else 'Tidak'}")
print("------------------------------------------")

if usia >= 60:
    if punya_rekomendasi == True:
        print("‚úÖ Pengecekan 1: Usia terpenuhi.")
        print("‚úÖ Pengecekan 2: Rekomendasi terpenuhi.")
        print("\nüéâ Selamat! Anda dinyatakan Layak.")
    else:
        print("‚úÖ Pengecekan 1: Usia terpenuhi.")
        print("‚ùå Pengecekan 2: Rekomendasi TIDAK terpenuhi.")
        print("\nüéâ Selamat! Anda dinyatakan Layak (karena usia).")
else:
    if punya_rekomendasi == True:
        print("‚ùå Pengecekan 1: Usia TIDAK terpenuhi.")
        print("‚úÖ Pengecekan 2: Rekomendasi terpenuhi.")
        print("\nüéâ Selamat! Anda dinyatakan Layak (karena rekomendasi).")
    else:
        print("‚ùå Pengecekan 1: Usia TIDAK terpenuhi.")
        print("‚ùå Pengecekan 2: Rekomendasi TIDAK terpenuhi.")
        print("\nMaaf, Anda tidak memenuhi syarat kelayakan.")

Program Menggunakan Pruning

Pendekatan ini lebih efisien. Jika usia sudah memenuhi syarat, program langsung memberikan hasil "Layak" tanpa perlu memeriksa surat rekomendasi (pruning).

# Inisialisasi variabel
usia = 65
punya_rekomendasi = False

print("\n--- Pengecekan Kelayakan (Dengan Pruning) ---")
if usia >= 60:
    print("‚úÖ Pengecekan 1: Usia terpenuhi. (Proses berhenti)")
    print("\nüéâ Selamat! Anda dinyatakan Layak.")
else:
    print("‚ùå Pengecekan 1: Usia TIDAK terpenuhi.")
    if punya_rekomendasi == True:
        print("‚úÖ Pengecekan 2: Rekomendasi terpenuhi.")
        print("\nüéâ Selamat! Anda dinyatakan Layak.")
    else:
        print("‚ùå Pengecekan 2: Rekomendasi TIDAK terpenuhi.")
        print("\nMaaf, Anda tidak memenuhi syarat kelayakan.")

Ketentuan Bonus:

  1. Bonus Kinerja (Logika AND): Seorang karyawan berhak mendapatkan Bonus Kinerja jika Penilaian Kinerja (skala 1-10) mereka bernilai di atas 8 DAN Kehadiran (dalam persen) mereka di atas 95%.

  3. Gunakan struktur Nested IF (tanpa prunning dan dengan pruning) untuk mengimplementasikan logika di atas. Jangan gunakan operator logika and atau or.
  4. Berikan output/print yang jelas untuk setiap langkah pengecekan dan hasil akhirnya (mirip seperti contoh pada materi ini).
